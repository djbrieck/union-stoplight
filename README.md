# union-stoplight (Un-official)

This is a custom unofficial site generated by the [Un‑official Static Website Generator](https://github.com/djbrieck/Un-official-static-website-generator).

## Build

Check out the generator repo and place this site’s folder as a sibling to it then run the following command from this projects folder:

```bash
python3 ../Un-official-static-website-generator/generate.py site.md 
``` 
This will read the `site.md` config file, generate `dist/index.html` and favicons, and copy any `css/`, `js/`, `img/`, or `docs/` folders into `dist/`.

## Edit the `site.md` file to customize the content of the generated site; re-run the build command to update the output.

## Run, locally from this folder after building to preview the generated live site: 

```bash
 python3 -m http.server 8000 --directory dist
```

Open http://localhost:8000 in a browser to preview the generated site.

## Deploy, finally the contents of the `dist/` folder to your hosting provider or [IPFS](https://ipfs.tech/).
